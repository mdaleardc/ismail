JavaScript Arithmetic Operators

Arithmetic Operators take numerical values as thier operand and return a single numerical value.

In JavaScript there are 8 basic arithmetic operators.

Operator Description
+ Addition
- Subtraction
* Multiplication
/ Devision
** Exponentiation
% Remainder
++ Increment
-- Decrement

Addition (+)

The addition operator ( + ) returns the sum of its numerical operands.

Subtraction (-)

The subtraction operator ( - ) returns the difference of its operands.

Multiplication (*)

The multiplication operator ( * ) returns product of its numerical operands.

Division (/)

The division operator ( / ) returns quotient of its numerical operands.

Where the left operand is the dividend and the right operand is the divisor.

Exponentiation (**)

The exponentiation operator ( ** ) returns the result of raising its first operands to the power of its second operands.

Increment (++)

The increment operator ( ++ ) add 1 to its operand and returns a value.

Decrement (--)

The decrement operator ( -- ) deduct 1 from its operand and returns a value

Operator Sequense

The operator sequence describes the order of performed operation in an arithmetic expression.

JS follows the MDAS pattern in math, where multiplication is performed first, division is second, addition is third and deduction deduction is fourth or last.


But what if we need certain operation to be performed first?

For example, what if we want addition to be performed first in our previous example?


We can use grouping to do that, grouped expressions are performed first before the others.

Use parentheses ( ) to group expressions.
