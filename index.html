Css3 Flexbox Align Content

Note:Since the align-content property does not work when there is only one line of flex items, we need to set the flex-wrap: wrap CSS declaration to wrap the flex items onto multiple lines.


align-content valid values:

  • stretch: flex items are stretched to take the remaining space
  • flex-start: packs items to the start of the flex container's cross start
  • flex-end: packs items to the start of the flex container's cross end
  • center: packs items to the center toward the center of the alignment of the cross axis
  • space-between: distributes flex items evenly within the alignment of the cross axis; the space between adjacent (any two) items is equal; the first line is at the start of the cross axis and the last is at the end
  • space-around: distributes flex items evenly within the alignment of the cross axis with equal space around each line
